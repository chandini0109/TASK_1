# 🧮 Calculator

## 📌 Project Overview
This project is a simple and responsive Calculator developed using HTML, CSS, and JavaScript as part of the AICTE Oasis Infobyte Web Development Internship (Level 2 - Task 1). The calculator performs basic arithmetic operations through an interactive and user-friendly interface.

## 🚀 Features
- Simple and responsive design
- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Percentage (%)
- Decimal calculations
- Clear (C) button
- Backspace (⌫) button
- Error handling for invalid expressions

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
Calculator/

```

## ▶️ How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Double-click the `index.html` file or open it in any web browser.
4. Start performing calculations.
