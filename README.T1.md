# 🌐 Landing Page

## 📌 Project Overview
This project is a simple and responsive landing page created using HTML and CSS. It is designed as part of the AICTE Oasis Infobyte Web Development Internship (Level 1 - Task 1). The page demonstrates the fundamentals of web design, including layout creation, styling, and responsive design.

## 🚀 Features
- Responsive and modern design
- Clean navigation bar
- Attractive hero section
- Call-to-action button
- Features section
- Footer section
- User-friendly interface

## 🛠️ Technologies Used
- HTML5
- CSS3

## 📂 Project Structure
```
Landing-Page/
│── index.html
│── README.md
```

## ▶️ How to Run
1. Download or clone the project.
2. Open the project folder.
3. Double-click the `index.html` file or open it in any web browser.
4. The landing page will be displayed.

## 📚 Learning Outcomes
- Understanding HTML page structure
- Styling web pages using CSS
- Creating responsive layouts
- Using Flexbox for alignment
- Designing modern user interfaces
